## 🗺️ GUÍA DE USO SEGÚN ROL

### Si eres un USUARIO NUEVO:
1. Lee el **Manual de Usuario** completo
2. Revisa las **Preguntas Frecuentes**
3. Explora la sección de **Solución de Problemas**

### Si eres ADMINISTRADOR:
1. Lee el **Manual de Administrador**
2. Familiarízate con el **Sistema de Analítica**
3. Revisa el **Manual de Usuario** para entender la experiencia del usuario

### Si eres DESARROLLADOR NUEVO:
1. Comienza con el **Manual de Instalación**
2. Lee el **Manual Técnico**
3. Revisa el **Manual de Firebase**
4. Estudia el **Manual de Despliegue**
5. Consulta los **Índices de Firestore**

### Si eres DEVOPS:
1. Lee el **Manual de Despliegue**
2. Revisa el **Manual de Firebase**
3. Familiarízate con las **Variables de Entorno**

## 📋 CONVENCIONES DE DOCUMENTACIÓN

### Formato de Archivos
- Todos los manuales están en formato **Markdown (.md)**
- Uso de emojis para mejor visualización
- Código con syntax highlighting
- Diagramas ASCII cuando sea necesario

### Estructura Estándar
```
# TÍTULO DEL MANUAL

## 📋 INFORMACIÓN GENERAL
## 🎯 SECCIÓN 1
## 🔧 SECCIÓN 2
...
## 📞 CONTACTO/SOPORTE
```

### Niveles de Encabezado
- `#` - Título principal
- `##` - Secciones principales
- `###` - Subsecciones
- `####` - Detalles específicos
